This the the formula section here we can easily access the formula to do different tasks in the worksheet.

# Action Formula Description
1 Sum =Sum(Ai : Aj) where 1 ≤ i < j ≤ n To get the sum of perticular category we use this formula.
2 Grand Total =Sum(Ai , Aj) where 1 ≤ i < j ≤ n To get the sum of categories we use this formula.
3 Copy p=Copy(Ai:Aj) where 1 ≤ i < j ≤ n and p can be anything To copy the value of one row into multiple rows of the same column use this formula.
4 Distribute p=Distribute(Ai.....Aj) where 1 ≤ i < j ≤ n and p should be integer To Distribute the value of the Grand total among the other rows on the basis of the ratio from the preceeding column use this formula
